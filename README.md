# CSE 341 Web Services API

## Description
This project is a RESTful API developed for the CSE 341 Web Services course. 
It demonstrates backend development using Node.js and database integration.

The API allows users to manage resources through standard CRUD operations 
(Create, Read, Update, Delete).

## Technologies Used
- Node.js
- Express
- MongoDB
- Render (deployment)
- Swagger API documentation

## Features
- REST API endpoints
- Database integration
- API documentation with Swagger
- Deployed cloud backend
